# Roadmap

Next features will be:

## Persistence

Save meetings to D1 whenever queried.

## iCal & RSS support

Add ical and rss endpoints that provide upcoming meetings in iCal and RSS formats, including query parameters to select which agencies.

## Agency to channel support

Add ability to output certain agencies to specific Slack channels.

## Slack: Just Added and Upcoming

"Meetings this week" feature (in one channel), and a "just added" feature (in another channel).
